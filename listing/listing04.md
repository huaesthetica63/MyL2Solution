Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
Программа завершится ошибкой deadlock, это объясняется тем, что в main будет ожидаться закрытие канала chan, но этого закрытия нигде нет. Для того, чтобы исправить эту ошибку, после цикла for в горутине нужно добавить close(ch) - после этого программа сможет удачно завершиться, а перед этим вывести последовательность чисел от 0 до 9, которые помещаются в канал ch внутри горутины и читаются в main. Конструкция range с чтением канала примечательна тем, что чтение продолжается до тех пор, пока канал открыт, то есть, в примере этот цикл будет бесконечным, поскольку канал открыт всегда. Вместо закрытия канала ошибку deadlock можно так же исправить заменой цикла range на такой же цикл, как в горутине: for i:=0; i<10; i++{
	n:=<-ch
	println(n)
}

```
